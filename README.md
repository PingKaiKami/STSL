# 遊戲人工智慧期末專案 (Game AI Final Project)

歡迎來到遊戲人工智慧期末專案！為確保團隊協作順暢與專案結構整潔，請所有成員在開發前詳閱以下環境建置與開發規範。

---

## 🚀 快速上手 (For Unity 初學者)

如果你是第一次開啟本專案，請按照以下步驟操作環境建置：

1. 透過 Git 將本專案 **Clone** 到你的本機電腦中。
2. 開啟 **Unity Hub**。
3. 點選右上角的 **Add (加入)** 按鈕。
4. 選擇你剛剛 Clone 下來的專案根目錄資料夾。
5. 等待 Unity 載入與編譯專案（第一次開啟專案會需要一些時間載入 Library，請耐心讓它跑完 ☕）。

---

## 🛠️ 開發與協作規範 (For 專案開發者)

為了避免 Git 版本衝突與檔案混亂，已經開啟專案準備實作的成員，請嚴格遵守以下規範：

### 1. 分支管理 (Branching)
🛑 **開始實作之前，請務必先建立並切換到自己的 Branch！** 請勿直接在主分支上進行修改與開發。

### 2. 場景管理 (Scenes)
* 請在 `Scenes` 資料夾下，建立一個**專屬於你自己的場景 (Scene)**，並在該場景中測試與實作你的內容。
* 💡 *小提示：你也可以直接複製專案中現有的 `Battle Scene`，將其重新命名為你的專屬場景後再進行修改。*

### 3. 程式碼管理 (Scripts)
* 所有程式碼請統一集中放置在 `Scripts` 資料夾內。
* **分類規範**：請在 `Scripts` 資料夾中，建立一個**與你的場景名稱相同**的子資料夾，再將你撰寫的腳本放進去，以利日後辨識與整合。

### 4. 資源管理 (Assets)
* 非程式碼的專案資源（例如：Prefabs, Materials, Animations 等），皆有其對應的專屬資料夾。請務必分類放置。
* 若你匯入或創建了新的資源，且目前沒有合適的分類，請自行新增一個標示清楚的資料夾。
