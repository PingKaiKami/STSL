# 遊戲人工智慧期末專案

## For 初次使用 Unity 的人

**非常不建議使用**

畢竟短短半個學期我沒有餘力再去處理意料外的錯誤

你只要告訴我功能 我就會盡可能實現給你

---

如果你真的非常堅持 那就

1. 先clone 
2. 然後開啟Unity Hub
3. 點選Add
4. 選擇你clone的資料夾
5. 讓他煮 !!!

## For 已經開啟專案的人

如果你要實作的話**請先自行branch**

然後在`Scenes`資料夾下創立屬於自己的場景

並在該場景做你實作的內容

---

程式碼統一建立在`Scripts`下

記得一樣再創一個自己 Scenes 名字的資料夾後再放

基本上非程式碼的東西都有自己專屬的資料夾

如果真的沒有就請自行新增新的上去
